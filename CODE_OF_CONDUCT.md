# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

In short: be respectful, welcoming, and constructive. Harassment, discrimination, and
personal attacks are not tolerated. Assume good faith, keep feedback about the work, and
help make this a place people are glad to contribute to.

The full text is at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

**Reporting:** to report unacceptable behavior, contact the maintainer
(**@edwardvaneechoud**) via a GitHub private message or by opening a confidential
[security advisory](https://github.com/edwardvaneechoud/flowfile-community-nodes/security/advisories/new)
if you'd rather it stay private. Reports are handled confidentially.
