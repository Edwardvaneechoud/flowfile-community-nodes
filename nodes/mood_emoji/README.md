# Mood Emoji

Add an emoji column derived from a numeric mood column. Pick a theme
(performance, temperature, money, energy, love, chaos, pizza), set a threshold,
and choose how much sparkle you want.

![Settings form](screenshots/form.png)

## Usage

1. Connect any source with a numeric column.
2. Point **Analyze This Column** at that column.
3. Pick an **Emoji Mood Logic** theme and a **Mood Threshold**.
4. Run — a new emoji column appears in the output.
